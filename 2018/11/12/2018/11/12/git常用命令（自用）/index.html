git常用命令

git常用命令（自用）

初次使用

  • 初始化一个git仓库

    git init

  • 保存所有更改

    git add .

  • 保存某文件的更改

    git add <file>

  • 提交以及理由

    git commit -m "叙述为什么提交"

分支操作

  • 同步到远程服务器的master分支（比如github）

    git push -u origin master

  • 查看所有分支

    git branch

  • 新建分支

    git branch <name>

  • 切换到name分支

    git checkout <name>

  • 创建并切换到name分支

    git checkout -b <name>

  • 合并name分支到当前分支

    git merge <name>

  • 删除分支

    git branch -d <name>

版本控制

  • 查看当前分支的所有提交

    git log

  • 回到某一版本（commit_id无需写全，至少写前几个字母、数字）

    git reset --hard <commit_id>

  • 回到上一个版本

    git reset --hard HEAD^

  • 回到上上一个版本

    git reset --hard HEAD^^

  • 查看最近操作

    git reflog

版本号

  • 新建版本号

    git tag <tagname>

  • 创建带说明的版本号

    git tag -a <tagname> -m " "

  • 查看当前所有版本号

    git tag

  • 为某次commit添加版本号

    git tag <tag><commit_id>

  • 详细查看某一个版本

    git show <tagname>
