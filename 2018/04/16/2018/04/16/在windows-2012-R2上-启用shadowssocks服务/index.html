在windows 2012 R2上 启用shadowssocks服务

在windows 2012 R2上 启用shadowssocks服务

Shadowsocks是一个轻量级隧道Socks5代理，可加密网络通道。

1、下载nodejs

首先要进入https://nodejs.org/，点击页面上的INSTALL安装相对应的node环境。

2、安装nodejs

很简单，一路点下一步。

3、安装ShadowSocks

打开“开始->所有程序->Node.js->Node.js command prompt”，打开nodejs命令端，输入命令：
npm install -g shadowsocks

4、设置ShadowSocks

首先找到shadowsocks安装目录，
也就是下面的这个目录C:\Users\Administrator\AppData\Roaming\npm\node_modules\shadowsocks，
然后找到config.json文件，编辑一下它。

 { 
"server":"你的内网ip", 
"server_port":端口号（一般是443就行）, 
"local_address":"127.0.0.1", 
"local_port":1080, 
"password":"密码", 
"timeout":600, 
"method":"加密算法"
}

5、运行shadowsocks

运行服务端 打开Node.js command prompt 在弹出的命令行中输入：	    
ssserver

6、接下来就是他的网络连接状态

注意：

Node.js command prompt不可以关，而且服务开启可能会导致服务器高负载！！
一切的完成都要在防火墙允许情况下进行（防火墙可关闭）
